Menções honrosas

  • Replicant AOSP - Um sistema operacional de código-fonte aberto baseado no Android, com o objetivo de substituir todos os componentes proprietários por software livre.
  • OmniROM AOSP contrib - Um sistema operacional de software livre para smartphones e tablets, baseado no Android.
  • MicroG Pacote de Extensões contrib - Um projeto que tem por objetivo reimplementar os Serviços Google Play proprietários no sistema operacional Android por um substituto FLOSS. O projeto microG também mantém um fork do LineageOS com microG e F-Droid pré-instalados em Lineage para microG.
