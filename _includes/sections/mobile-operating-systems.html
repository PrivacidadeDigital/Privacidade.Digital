Menções honrosas

  • Replicant - Um sistema operacional de código aberto baseado no Android, com o objetivo de substituir todos os componentes proprietários por software livre.
  • OmniROM - Um sistema operacional que é software livre, para smartphones e tablets e baseado no Android.
  • MicroG - Um projeto queu tem por objetivo reimplementar os Serviços Google Play proprietários no sistema operacional Android com um substituto FLOSS.
