Menções honrosas

  • RetroShare - Um software cliente de mensagens instantâneas encriptadas com E2E e chamadas de vídeo/voz. RetroShare também suporta TOR e I2P.
  • ChatSecure - ChatSecure é um app de mensagens gratuito e de código aberto que tem recursos de encriptação OTR sobre XMPP.
  • Kontalk - Uma rede de mensagens instantânea dirigida pela comunidade. Suporta encriptação ponta a ponta. Tanto os canais do cliente com o servidor quanto o servidor com o servidor são totalmente encriptados.
  • Conversations - Um cliente Jabber/XMPP de código aberto para smartphones com Android 4.4 ou superior. Suporta encriptação de ponta a ponta com OMEMO ou openPGP. Também existe o Conversations Legacy , que ainda suporta OTR.
  • Wire experimental (mais info) - Um software livre de chats encriptados de ponta a ponta que suporta mensagens e chamadas de vídeo e voz.
  • Status - Experimental Serviço encriptado de mensagens instantâneas com código aberto, P2P, e com suporte a DAPPs.

Atualizando o programa Tor incluído no Ricochet

Atualmente não existem vulnerabilidades conhecidas no Ricochet. O software é seguro de usar contant que você atualize o programa Tor incluído para sua versão mais nova.
  1. Download the Tor Browser.
  2. Run the Tor Browser installer or extract package for your platform. Windows (.exe), MacOSX (.dmg) and Linux (.tar.bz2).
  3. Copy the Tor binary or create a symbolic link to it:

    • Windows: Copy the Tor binary included with the Tor Browser and overwrite the old one included with Ricochet.
      • Tor Browser binary: %HOMEPATH%\Desktop\Tor Browser\Browser\TorBrowser\Tor\tor.exe
      • Ricochet's included binary: %HOMEPATH%\Desktop\Ricochet\tor.exe

    • MacOSX: Assuming you copied the Tor Browser.app and Ricochet.app from their respective .dmg install media to /Applications:
      Create a symbolic link to the Tor binary in Terminal. Ricochet will continue to use the up to date version of the Tor binary when you update the Tor Browser Bundle.
      • rm ~/Applications/Ricochet.app/Contents/MacOS/tor
        ln -s ~/Applications/Tor\ Browser.app/Contents/Resources/TorBrowser/Tor/tor \
        ~/Applications/Ricochet.app/Contents/MacOS/tor

    • Linux: Assuming you extracted the Tor Browser Bundle and Ricochet tarball to your home directory:
      • rm ~/ricochet/tor
        ln -s ~/tor-browser_en-US/Browser/TorBrowser/Tor/tor ~/ricochet/tor
