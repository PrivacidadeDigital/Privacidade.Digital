Alerta

  • Nota: A partir de dezembro de 2018, o Joplin não suporta proteção por senha/pin para o aplicativo em si ou notas/cadernos individuais. Os dados ainda são criptografados em trânsito e no local de sincronização usando sua chave mestra. Veja o assunto em aberto.

Menções honrosas

  • Notable - O aplicativo de anotações baseado em marcadores que não é uma porcaria.
  • Paperwork - Solução de código aberto e auto-hospedagem. Para servidores PHP / MySQL.
  • Org-mode - Um modo principal para o GNU Emacs. Org-mode é para fazer anotações, manter listas de afazeres, planejar projetos e criar documentos com um sistema de texto puro rápido e eficaz.
