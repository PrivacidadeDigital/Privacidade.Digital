Extensões de Privacidade para iOS

Use uma VPN com o Passepartout

Passepartout é uma aplicativo OpenVPN amigável gratuíto e de código-fonte aberto com predefinições para vários provedores (incluindo Mullvad e ProtonVPN, dois provedores recomendados pelo Privacidade.Digital) e facilmente configurável.

Block content in Safari with AdGuard for iOS

AdGuard for iOS is a free (and open-source) content-blocking app that allows you to set filters within Safari. As required by iOS, it uses the native Content Blocker API for even greater privacy.
