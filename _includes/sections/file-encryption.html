  • Cryptomator - Encriptação AES gratuita feita no cliente para seus arquivos na nuvem. Software com código-fonte aberto: sem backdoors, sem registros.
  • DiskCryptor - Um sistema de encriptação para HDs inteiros ou partições no Windows. Inclui a habilidade de encriptar a partição em que o SO está instalado.
  • Linux Unified Key Setup (LUKS) - Um sistema de encriptação de HDs para Linux usando dm-crypt como o backend da encriptação do disco. Incluído por padrão no Ubuntu (mas não ativado por padrão). Disponível também para Windows e várias distribuições GNU/Linux.
  • Hat.sh - Uma aplicação web JavaScript e multi-plataforma que provém encriptação segura de arquivos usando o algoritmo AES-256-GCM no seu navegador. Também pode ser baixado e rodado offline.
  • Keka - Um compressor de arquivos disponível somente para macOS, com código-fonte aberto e funcionalidade de encriptação de arquivos.
    • =======
  • Cryptomator - Free client-side AES encryption for your cloud files. Open-source software: No backdoors, no registration.
  • DiskCryptor - A full disk and partition encryption system for Windows including the ability to encrypt the partition and disk on which the OS is installed.
  • Linux Unified Key Setup (LUKS) - A full disk encryption system for Linux using dm-crypt as the disk encryption backend. Included by default in Ubuntu. Available for Windows and Linux.
  • Hat.sh - A cross-platform, serverless JavaScript web application that provides secure file encryption using the AES-256-GCM algorithm in your browser. It can also be downloaded and run offline.
  • Keka - A macOS-only, open-source file archiver with the ability to encrypt files.
