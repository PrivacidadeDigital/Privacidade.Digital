Menções Honrosas

  • Cryptomator - Encriptação AES gratuita feita no lado do cliente para seus arquivos na nuvem. Software open source: Sem backdoors, sem registros.
  • DiskCryptor - Um sistema de encriptação para discos completos ou partições para Windows, inclui a abilidade de encriptar a partição em que o SO está instalado.
  • Linux Unified Key Setup (LUKS) - Um sistema de encriptação de disco completo para Linux usando dm-crypt como o backend da encriptação do disco. Incluído como padrão no Ubuntu. Disponível também para Windows e Linux.
