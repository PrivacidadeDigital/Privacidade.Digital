Menções Honrosas

  • Cryptomator - Encriptação AES gratuita feita no cliente para seus arquivos na nuvem. Software com código-fonte aberto: sem backdoors, sem registros.
  • DiskCryptor - Um sistema de encriptação para dispositivos de armazenamento inteiros ou partições no Windows. Inclui a habilidade de encriptar a partição em que o SO está instalado.
  • Linux Unified Key Setup (LUKS) - Um sistema de encriptação de dispositivos de armazenmento para Linux usando dm-crypt como o backend da encriptação do dispositivo. Incluído por padrão no Ubuntu (mas não ativado por padrão). Disponível também para Windows e várias distribuições GNU/Linux.
  • Hat.sh - Uma aplicação web JavaScript e multi-plataforma que provém encriptação segura de arquivos usando o algoritmo AES-256-GCM no seu navegador. Também pode ser baixado e rodado offline.
  • Keka - Um compressor de arquivos disponível somente para macOS, com código-fonte aberto e funcionalidade de encriptação de arquivos.
