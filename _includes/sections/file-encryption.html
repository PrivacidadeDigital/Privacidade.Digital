Menções honrosas

  • Cryptomator - Encriptação AES gratuita feita no cliente para seus arquivos na nuvem. Software de código-fonte aberto: sem backdoors, sem registros.
  • DiskCryptor - Um sistema de encriptação para discos completos ou partições para Windows. Inclui a habilidade de encriptar a partição em que o SO está instalado.
  • Linux Unified Key Setup (LUKS) - Um sistema de encriptação de disco completo para Linux usando dm-crypt como o backend da encriptação do disco. Incluído por padrão no Ubuntu (mas não ativado por padrão). Disponível também para Windows e outras distribuições GNU/Linux.
